# SSHMAP(1)

## NAME

sshmap - agentless SSH exposure management CLI

## SYNOPSIS

**sshmap** *command* \[*options*\]

## DESCRIPTION

SSHMap discovers SSH services, collects read-only configuration evidence, analyzes risks, and builds an access graph backed by SQLite.

## COMMANDS

**init**, **doctor**, **discover**, **scan**, **local-scan**, **analyze**, **report**, **serve**, **export**

See **sshmap --help** for the full command tree.

## SAFETY

Only use SSHMap against systems you own or are explicitly authorized to assess.
